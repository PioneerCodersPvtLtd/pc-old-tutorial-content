Write a program on parameterized constructors?
public class Studentdetail { int id; String name; Studentdetail(int id, String name) { this.id = id; this.name = name; } void displayData() { System.out.println(id + " " + name); } public static void main(String args[]) { Studentdetail student1 = new Studentdetail(111, "Arun"); Studentdetail student2 = new Studentdetail(222, "Varun"); student1.displayData(); student2.displayData(); } }
Write a program on constructor-Overloading?
public class Persondetail { int id; String name; int age; Persondetail(int id, String name) { this.id = id; this.name = name; } Persondetail(int id, String name, int age) { this.id = id; this.name = name; this.age = age; } void display() { System.out.println(id + " " + name + " " + age); } public static void main(String args[]) { Persondetail person1 = new Persondetail(111, "Karan"); Persondetail person2 = new Persondetail(222, "Aryan", 25); person1.display(); person2.display(); } }
