Write a program for natural numbers using WhileLoop?
public class WhileLoopExample { public static void main(String args[]) { int size = 4; int i = 1; while (i <= size) { System.out.println(i); i++; } } }
Write a program for displaying elements using For-Each Loop??
public class FirstForEachLoop { public static void main(String args[]) { int[] array = { 1, 3, 5, 7, 6 }; for (int i : array) { System.out.println(i); } } }
Write a example programm on NestedForLoop?
public class StarPattern { public static void main(String[] args) { int n = 2; for (int i = 1; i <= n; i++) { for (int j = 1; j <= i; j++) { System.out.print("*"); } System.out.println(); } } }
