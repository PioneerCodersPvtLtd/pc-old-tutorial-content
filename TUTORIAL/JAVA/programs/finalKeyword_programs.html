Write a program with final keyword
class Bikeandcar { final int speedlimit=90;//final variable void run(){ speedlimit=400; } public static void main(String args[]){ Bikeandcar obj=new Bikeandcar(); obj.run(); } }
Example for initialising blank final variable
class Speeding { final int speedlimit;//blank final variable Speeding(){ speedlimit=70; System.out.println(speedlimit); } public static void main(String args[]){ new Speeding(); } }
