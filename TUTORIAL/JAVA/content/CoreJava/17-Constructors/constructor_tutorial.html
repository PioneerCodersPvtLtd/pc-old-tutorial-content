What is constructor ?

  • A java constructor has the same name as the name of the class to which it belongs.
  • Constructor’s syntax does not include a return type, since constructors never return a value.
  • It is invoked at the time of object creation with new operator.
  • If there is no constructor in a class, compiler automatically creates a default constructor.
  • Class can have any number of constructors that differ in parameter lists.

Why we need constructor ?

Constructor is used to initialize the instance variables values at the time of object creation.

Types of java constructors

There are two types of constructors:

1.Default constructor (no-arg constructor)

2.Parameterized constructor
