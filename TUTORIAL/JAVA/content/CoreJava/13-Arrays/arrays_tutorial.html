What is Array

In Maths, An arrangement of objects, pictures, or numbers in columns and rows is called an array.

Similarly in Java, An array is group of memory locations allocated in side by side to store similar type of data.

@IMG_START@JAVA/array/gif@IMG_END@
  • Each item in an array is called an element
  • each element is accessed by its numerical index.
  • As shown in the preceding illustration, index numbering begins with 0. ends with array length-1.
  • It store the value on the basis of index value.
  • It is fixed in size that means you can't increase the size of array at run time.

Declaring Array Variables:

To use an array in a program, you must declare a variable to reference the array, and you must specify the type of array the variable can store.

Syntax
@CODE_START@@JAVA@dataType[] arrayVar; (or) dataType arrayVar[]; int[] myList; (or) int myList[]; // declaration myList = {10,20,30}; // instantiation dataType[] arrayVar = {value0,value1,........,valuek}; // declaration and instantiation. int arr[] = {10,20,30}; // declaration and instantiation. @CODE_END@

Note : The style dataType[] arrayVar is preferred. The style dataType arrayVar[] comes from the C/C++ language and was adopted in Java to accommodate C/C++ programmers.

At the time of array declaration we can not specify the size of array. For Example int[5] arrayVar; this is wrong.

Creating Arrays with new keyword:

You can create an array by using the new operator with the following syntax:

@CODE_START@@JAVA@ datatype[] arrayVar = new dataType[arraySize];@CODE_END@

The above statement does two things:

  • It creates an array using new dataType[arraySize];
  • It assigns the reference of the newly created array to the variable arrayVar;

Assigning values to array

@CODE_START@@JAVA@arrayVar[index] = value;@CODE_END@

There are two types of array in java they are

1).Single Dimensional Array

2).Multidimensional Array

Sample example on Single Dimensional array
class ArrayEx { public static void main(String[] args){ int arr[] = {10,20,30}; arr[0] = 40; arr[1] = 50; arr[2] = 60; System.out.println(arr[0]); System.out.println(arr[1]); System.out.println(arr[2]); } }

Note:

1)If we specify array size as negative int value, then we will get run-time error, NegativeArraySizeException

2) To specify array size the allowed data types are byte, short, int, char If we use other data type then we will get an Compile time error.

3) The maximum allowed size of array in java is 2147483647 (It is maximum value of int data type)

Processing Arrays using for loop:

When processing array elements, we often use either for loop or foreach loop because all of the elements in an array are of the same type and the index is start with 0 and ends with length-1. By using for loop we will generate index to acces array element at index.
