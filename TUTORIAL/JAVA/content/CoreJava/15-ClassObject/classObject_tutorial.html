Object

An object is an instance of a class or object is creation of memory structure for class.

Characteristics of an object.

Objects have state, behaviors and identity.

  • State : represent the data(values) of an object.
  • Behavior : represent the behavior (functionality's ) of an object.
  • Identity : every object in this world has identity to differentiate with others object.
Example:
A Student has properties- id, name and email etc. behaviors - reading, writing and studing etc.

Let us now look deep into what are objects. If we consider the real-world we can find many objects around us, Cars, Dogs, Humans, etc. All these objects have a properties, behavior and state.

If you compare the software object with a real world object, they have very similar characteristics.

Software objects also have a state and behavior. A software object's state is stored in fields and behavior is shown via methods.

So in software development, methods operate on the internal state of an object and the object-to-object communication is done via methods.

Creating an Object :

As mentioned previously a class provides the blueprints for objects. So basically an object is created from a class. In java the new key word is used to create new objects.

There are three steps when creating an object from a class:

Declaration A variable declaration with an object type.

Instantiation The 'new' key word is used to create the object.

Initialization The 'new' keyword is followed by a call to a constructor. This call initializes the new object.

Object Creation Syntax
@CODE_START@@JAVA@/* First create an object */ ClassName ObjectReference = new Constructor(); @CODE_END@

Accessing Instance Variables and Methods :

Instance variables and methods are accessed via (.) dot operator on object reference.

Syntax
@CODE_START@@JAVA@/* Now call a variable as follows */ ObjectReference.variableName; /* Now you can call a class method as follows */ ObjectReference.methodName(); @CODE_END@
