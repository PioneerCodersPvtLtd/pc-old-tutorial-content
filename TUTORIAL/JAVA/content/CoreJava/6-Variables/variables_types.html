There are three types of variables in java

  • local variable
  • instant variable
  • static variable

Local variable

A variable that is declared inside the method is called local variable.

instant variable

A variable that is declared inside the class but outside the method is called instance variable(class level variables) . It is not declared as static.

static variable

A variable that is declared as static is called static variable. It cannot be local.

Syntax

@CODE_START@@JAVA@ class A{ int x=10; // Instance variable static int m=100; // static variable void method{ int n=90; // local variable } } @CODE_END@
