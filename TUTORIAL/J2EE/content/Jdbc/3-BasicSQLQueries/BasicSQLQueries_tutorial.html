SQL follows some unique set of rules and guidelines called Syntax. Some basic guidelines related to SQL Syntax are given below;

1) SQL is not case sensitive. Commonly SQL keywords are written in uppercase.

2) You can write SQL statements in one line or in multiple lines.

3) SQL are depends on relational algebra and tuple are relational calculus.

SQL statement

SQL statements are started with any of the SQL commands/keywords like SELECT, INSERT, UPDATE, DELETE, ALTER, DROP etc. and the statement ends with a semicolon (;).

Syntax

@CODE_START@@SERVLETS@SELECT * FROM table_name; @CODE_END@

Some Basic Commands

SELECT: It is used for Retrive data from a database

UPDATE: It is used for updates data in database.

DELETE: It is used for deletes data from database.

CREATE TABLE: It is used for creates a new table.

ALTER TABLE: It is used to modify the table.

DROP TABLE: It is used for deletes a table.

CREATE DATABASE: It is used for creates a new database

ALTER DATABASE: It is used to modify a database.

INSERT INTO: It is used for inserts new data into a database.

CREATE INDEX: It is used to create an index (search key).

DROP INDEX: It is used for deletes an index.

Create table

Syntax

@CODE_START@@SERVLETS@CREATE TABLE table_name ( column_name1 data_type(size), column_name2 data_type(size), column_name3 data_type(size), ............. );@CODE_END@

Example

@CODE_START@@SERVLETS@ CREATE TABLE Employee ( emp_id number(5), name varchar(20), salary number(10) );@CODE_END@

Employee table

emp_id name salary

SQL Select Command

Select command is used to retrieved the data from the existing table in the database.Using Select command you can retrieve specific records from the table "using where clause"

Syntax

@CODE_START@@SERVLETS@select * from table_name; @CODE_END@

Note: In the above syntax * represent all columns.

Example 1

@CODE_START@@SERVLETS@select * from employee; @CODE_END@

Select command with where clause

When you need to select some specific record from table you can use where clause with select command, in below example i will select those employee records who have more than 30000 salary.

Example

@CODE_START@@SERVLETS@SELECT * from Employee WHERE salary>30000; @CODE_END@

SQL Insert Command

Insert Command is used to insert new record in table. Using Insert Command you can insert a single or a multiple records in a table.Here tow ways to insert the data into database.

Insert data without speficying column name

Syntax

@CODE_START@@SERVLETS@INSERT INTO table_name VALUES (value1,value2,value3,...); @CODE_END@

Insert data by specifies both the column names and the values

Syntax

@CODE_START@@SERVLETS@INSERT INTO table_name (column1,column2,column3,...) VALUES (value1,value2,value3,...); @CODE_END@

Example

INSERT INTO Employee VALUES (101, 'amit', 24000);

Result after Execute above query.

emp_id name salary
101 Amit 24000

SQL Delete Command

Delete command is used to delete the record from the table.

Using Delete command you can delete all the records from the table without delete table.

Using Delete command you can delete specific records from the table "using where clause"

Syntax

DELETE FROM table_name WHERE some_column=some_value;

Example

DELETE FROM Employee WHERE salary<30000;

Result after Execute above query.
