Many to One Mapping :

1.Many to many is nothing but applying one to many relationship on both sides.Ex: one student can join in multiple courses and one course can contain multiple students.

2.Here we need collection property in both pojo classes and we need to configure this collection details in both mapping files.

3.While configuring collection property details we need to use to map the relationship.

Many to Many Mapping :

1. Many to many is nothing but applying one to many relationship on both sides.Ex: one student can join in multiple courses and one course can contain multiple students.

2. Here we need collection property in both pojo classes and we need to configure this collection details in both mapping files.

3. While configuring collection property details we need to use to map the relationship.

Note :In many to many relationship a mediator table is mandatory. We can call this table as join table. This join table contains only foreign keys.
