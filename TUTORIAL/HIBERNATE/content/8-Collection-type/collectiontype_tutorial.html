Collection in Hibernate

A collection is defined as a one-to-many reference. The simplest collection type in Hibernate is <list>.This collection is a list of unordered objects and can contain duplicates. This is similar to java.util.List.

Different Collection Types in Hibernate

java.util.List

java.util.Set

java.util.SortedSet

java.util.Map

java.util.SortedMap

java.util.Collection

or write the implementation of org.hibernate.usertype.UserCollectionType

List Type Collection

If our persistent class has List object, we can map the List easily either by element of class in mapping file or by annotation.

Ex Syntax @CODE_START@@HIBERNATE@ @CODE_END@

Set Type Collection

Set collection allows only the unique Objects to the collections.<set><set> element is used for mapping properties of type Set.
